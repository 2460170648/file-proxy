文件代下载服务

如要下载github项目，点击项目 Code 按钮，在 Download Zip 按钮上右键复制链接

文件支持任意格式，只要是文件直链均可以下载，支持命令行代下(详见项目说明)

所有下载节点均位于海外，可加速下载国外文档、资料，若速度不佳可切换下载方式

推荐 CloudFlare 代下，无速度及文件大小限制 。本站服务器代下限制 1024M
